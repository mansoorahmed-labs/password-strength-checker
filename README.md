# 🔐 Password Strength Checker

A Python-based Password Strength Checker that evaluates password strength using common cybersecurity best practices.

## Features

- ✅ Checks minimum password length
- ✅ Validates uppercase letters
- ✅ Validates lowercase letters
- ✅ Validates numbers
- ✅ Validates special characters
- ✅ Detects spaces in passwords
- ✅ Provides security recommendations
- ✅ Displays password strength

---

## Technologies Used

- Python 3
- Regular Expressions (Regex)

---

## Installation

Clone the repository

```bash
git clone https://github.com/mansoorahmed-labs/password-strength-checker.git
```

Go to the project

```bash
cd password-strength-checker
```

---

## Usage

Run the program

```bash
python3 password.py
```
